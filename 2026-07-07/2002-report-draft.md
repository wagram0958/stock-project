# 中鋼 (2002) 研究報告

**日期：** 2026-07-07

**收盤價：** 18.60 元（2026/07/07，-2.11%）

---

## 1. 摘要

中鋼（2002）為台灣唯一具備高爐一貫作業能力的鋼廠，是典型的景氣循環股。公司正從 2024-2025 年的營運谷底回升：盤價自 2026 年初已連漲 5-6 個月（2020 年以來最長紀錄）、4 月營收重返 300 億大關創 23 個月新高，法人預估 Q2 可望本業轉虧為盈。目前股價 18.60 元，股價淨值比（P/B）約 0.96 倍，低於歷史均值 1.3-1.4 倍，處於相對低檔。然而，2026 Q1 仍呈虧損狀態（EPS 數據有矛盾），且全年 EPS 法人預估中位數僅 0.09 元，景氣復甦力道仍待確認。

## 2. 公司概況

### 基本資料
| 項目 | 內容 |
|------|------|
| 成立/上市 | 1971 年成立，台灣上市（TWSE:2002） |
| 總部 | 高雄市前鎮區（臨海工業區） |
| 資本額 | 約 1,577 億元 |
| 產業 | 鋼鐵（一貫作業高爐煉鋼） |
| 市值 | 約 2,889 億元 |

### 業務範圍
- **核心業務：** 碳鋼扁鋼胚、熱軋鋼捲/板、冷軋鋼捲、電磁鋼片、鍍鋅鋼片等
- **下游應用：** 汽車、造船、建築、家電、3C、風電、軍工
- **高附加價值產品：** 高強度汽車用鋼（SM570 系列）、電磁鋼片、風電用鋼

### 競爭優勢
1. **台灣唯一高爐一貫作業鋼廠**，國內市佔率具寡佔地位
2. **垂直整合**：從煉鐵、煉鋼到軋鋼完整產線
3. **政府持股**：經濟部為最大股東，具政策庇護色彩
4. **碼頭與港口優勢**：緊鄰高雄港，進出口運輸成本低
5. **高附加價值產品轉型**：汽車鋼板、電磁鋼片訂單能見度至 2026 年中

### 結構性挑戰
- **中國鋼鐵產能過剩**：全球低價傾銷壓力巨大
- **美國 232 關稅**：升至 50%，壓縮出口空間
- **碳費趨勢**：高爐碳排高，未來碳費可能成為負擔
- **景氣高度循環**：獲利波動極大

## 6. Reviewer 審查意見（🔴 Claude-style Reviewer）

### 信心評分：42 / 100

### 邏輯與論證品質（6/10）
- 「谷底回升」論點與 Q1 EPS 矛盾、法人預估僅 0.09 元、7月盤價回調三大不確定因素衝突
- 「買在虧損期」邏輯過度簡化——未論證 2026-2027 有何等量級的需求催化劑
- 三情境估值矩陣缺乏機率權重，樂觀情境 P/B 應用歷史高峰而非均值
- 「相對抗跌」（大盤-2.31% vs 中鋼-2.11%）差異 0.2pp 無統計意義

### 資料來源檢驗（5/10）
- 🔴 **Q1 EPS 0.36 vs -0.16 矛盾未解決**——致命缺陷
- 🟡 法人預估 0.09~0.42 元差距 4.7 倍，來源不明
- 🟡 營收數據僅至 4 月，5-6 月已可取得但缺失
- 🟡 P/B 歷史均值未說明計算期間

### 魔鬼代言人：五大反面論點
1. **假底風險**：中國房地產長期下行→鋼鐵內需萎縮→過剩產能外銷，高爐技術面臨電弧爐取代，結構性衰退可能才剛開始
2. **量價背離**：盤價連漲但 1-4 月累計營收仍年減 2.96%，漲價可能只是成本轉嫁而非需求復甦
3. **價值陷阱**：P/B < 1 同時是價值陷阱經典信號——ROE 約 2% 遠低於資金成本 8-10%，日本新日鐵、韓國浦項 P/B 曾長期低於 1
4. **外資 4.2 萬張賣超是 Smart Money 離場**：聰明錢基於全球模型系統性看空，官股買超可能只是政策護盤
5. **殖利率 1.6% 毫無吸引力**：定存 1.5-1.8%、美元 >4%，持有中鋼的機會成本極高

### 認知偏誤警示
| 偏誤 | 表現 |
|------|------|
| 錨定效應 | 錨定「P/B 均值 1.3-1.4x」和「目標價 22 元」 |
| 確認偏誤 | 正面因子放大、負面因子輕描淡寫 |
| 近因偏誤 | 單月營收新高被當趨勢反轉 |
| 框架效應 | 「買在虧損期」框架省略了須確認景氣反轉的前提 |
| 倖存者偏差 | 引用成功案例忽略全球鋼鐵業重組/倒閉案例 |
